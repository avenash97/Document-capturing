# Document-capturing
Efficient detection and segmentation of document regions.

### Dataset description
Dataset comprises of document's captured images with noise. 

### Packages Required
```
1. sklearn
2. numpy
3. pandas
4. CV2
5. matplotlib
```
### Execute training python code as 
  ```
   $ jupyter notebook detect.ipynb  ```
or
  ```
   $ python detect.py
  ```
